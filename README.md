# ARK-Modded
Simple to set up dedicated ARK server with a personal selection of mods.

## Installed Mods
* Structures Plus (Mod-ID: 731604991)
* Bridges (Mod-ID: 558651608)

## Usage
Use exactly as the [vanilla ark image](https://github.com/Hetsh/docker-ark).
